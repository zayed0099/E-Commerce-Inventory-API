# E-Commerce-Inventory-API

#### Building a Inventory api to be used as a internal tool to manage a E-commerce Business.

Features that are nearly finished :
- Three roles: admin, support and packaging.
	-- **admin :**  Can manually manage the product stock, product data and so on.
	-- **support :**  Can watch all the products stock, see users order and communicate with them to provide support regarding their orders. Can't edit anything.
	-- **packaging :**  Mainly for warehouse usage. Someone with packaging role will only be able to watch the order's placed, stock of product and only manage is pacakging done or not.
